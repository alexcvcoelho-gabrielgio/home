Agilidade para salvar vidas!

O projeto é destinado a criar um sistema em que veículos que estejam trafegando no trânsito consigam trocar informações entre si e principalmente entre unidades de emergência, como ambulâncias e bombeiros, de forma que estes possam emitir alertas aos veículos que estejam em sua rota para que abram caminho, tornando mais ágil o atendimento às vítimas. Além da troca de informações entre veículos, é possível que estes enviem dados para a infraestrutura, que pode vir a ser uma central de monitoramento, a qual pode gerar dados para tomada de decisões utilizando o conceito de tecnologias Big Data.
